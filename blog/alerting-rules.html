Alerting rules

Back from summer vacation! Two weeks in Greece and two at home.

I've added guides for windows event alerting rules. Both are simple alerting rules, checking for an event in an eventlog and creates an alert when it's found. One of them checks for repeating events, when a number of events is genereated during a specified time frame the rule creates an alert.
